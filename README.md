# 🎬 Netflix Movie Analysis

This project focuses on analyzing Netflix movie data to extract meaningful insights using Python and data analysis libraries. The goal is to explore trends, patterns, and key factors that influence content on Netflix.

## 🚀 Features
- Data cleaning and preprocessing using Pandas
- Exploratory Data Analysis (EDA)
- Visualization of trends using Matplotlib/Seaborn
- Analysis of:
  - Movie genres distribution
  - Release year trends
  - Ratings and popularity
  - Duration and content type

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Project Structure
- `mymoviedb.csv` → Dataset used for analysis
- `Netflix movie analysis.ipynb` → Jupyter Notebook with complete analysis

## 📊 Key Insights
- Identified most popular genres on Netflix
- Analyzed growth of content over the years
- Compared ratings across different categories
- Observed trends in movie durations and types

## 🎯 Objective
To gain insights from Netflix data and demonstrate data analysis and visualization skills for real-world datasets.
